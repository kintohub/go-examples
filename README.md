
# Go Sample App

## :rocket: Deploying on Kintohub v1

### # Using Dockerfile 

- **Type:** Web App
- **Repo:** https://github.com/kintohub/go-examples
- **Branch:** master
- **Language:** Dockerfile
- **Dockerfile name:** Dockerfile
- **Subfolder Path:** `.`
- **Port:** 80
